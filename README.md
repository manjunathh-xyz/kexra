# Kexra

A small, expressive scripting language built for learning and experimentation.

## Installation

```bash
npm install -g kexra
```

## Example

```kx
say "Hello, Kexra"

fn add(a, b) {
  return a + b
}

say add(2, 3)
```

## What is Kexra?

- Interpreted scripting language
- Built in TypeScript
- Runs on Node.js
- Open source

## What Kexra is not

- Not production-ready
- Not a JavaScript replacement
- Not a framework
