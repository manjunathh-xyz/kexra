# math-extra

Extra math utilities for Kexra.

## Features

- Trigonometric functions (sin, cos)
- Mathematical constants (e)

## Example

```kx
use math-extra { sin, cos, e }

say sin(0)  # 0
say cos(0)  # 1
say e       # 2.718...
```
