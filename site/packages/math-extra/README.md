# math-extra

Extra math utilities for Kexra.

## Installation

```bash
kex install ./packages/math-extra
```

## Usage

```kx
use math-extra { sin, cos, e }

say sin(0)  # 0
say cos(0)  # 1
say e       # 2.718...
```

## API

- `sin(x)` - Sine of x (radians)
- `cos(x)` - Cosine of x (radians)
- `e` - Euler's number
